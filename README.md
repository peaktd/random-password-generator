## Описание

Random Password Generator - это графическое приложение для генерации безопасных случайных паролей с возможностью настройки параметров и сохранения истории генерации.

### Возможности

- ✅ Генерация случайных паролей с настраиваемой длиной (4-32 символа)
- ✅ Выбор типов символов:
  - Строчные буквы (a-z)
  - Заглавные буквы (A-Z)
  - Цифры (0-9)
  - Специальные символы (!@#$%^&*)
- ✅ Визуальный ползунок для выбора длины пароля
- ✅ Копирование пароля в буфер обмена
- ✅ История всех сгенерированных паролей
- ✅ Сохранение истории в JSON файл
- ✅ Автоматическая загрузка истории при запуске
- ✅ Проверка корректности ввода

### Технологии

- Python 3.x
- tkinter (GUI)
- random (генерация паролей)
- json (сохранение истории)
